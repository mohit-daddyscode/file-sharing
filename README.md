# file-share
A Spring Boot web app for uploading and downloading files

**Current functionalities:**
- *Folder*
  - Insert
  - Update
  - Delete
- *File*
  - Insert
  - Update
  - Delete
  - Download
- *Dynamic icon change*
- *User accounts*

**Functionality to be added:**
  - *Cache control*
  - *Multiple file sharing*
  - *Managing upload size*
  - *Permission*
